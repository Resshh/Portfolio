Gesture Controlled Virtual Mouse

A computer vision project that allows users to control the mouse cursor using hand gestures instead of a physical mouse. The system detects hand movements through a webcam and performs cursor movement, left click, right click, scrolling, and drag using real-time hand tracking.

Technologies:
Python, OpenCV, MediaPipe, PyAutoGUI
